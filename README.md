# The Use of Analogue Photographic Documentation to Infer the Past Appearance of Paintings

This repository contains the code and data developed for a Master’s thesis (Jan–Sep 2025) focused on **reconstructing the original appearance of paintings using archival slide film photographs**. The project is based on a novel dataset and combines **analogue photographic documentation, artificial ageing, colour correction, and image analysis** to detect pigment changes over time and support digital restoration efforts.  

The full thesis is available through NTNU’s database: [https://nva.sikt.no/registration/0199c398b026-a2da028d-a09e-4961-9651-6eb58addea0f](https://nva.sikt.no/registration/0199c398b026-a2da028d-a09e-4961-9651-6eb58addea0f)


## Overview

Paintings change over time due to ageing, environmental exposure, and conservation history. This project explores how **analogue photographic records** of artworks (e.g., archival slide film) can be used to **reconstruct earlier appearance states** of paintings and to quantify **pigment and colour changes** between past and present. :contentReference[oaicite:1]{index=1}

The methods combine:
- **Spectral imaging** and reflectance measurements
- **Colour correction techniques** to normalise archival photographs
- **Comparison and change detection** between historical and contemporary imagery

## Key Features

- Tools for **colour correction** of scanned archival slides and film
- Methods for **aligning and comparing images** across time
- Support for analysing **perceptual colour changes** that can indicate material alteration

## Structure

- `code/` – Scripts and functions for colour correction, calibration, and image processing  
- `data/` – Example images and reference scans (archival slides and contemporary captures)  
- `results/` – Outputs from processing pipelines and comparison visualisations
